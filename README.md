# 喵记 — 治愈系动画日记

一只陪伴你记录每天心情的虚拟猫咪日记应用。

## 简介

喵记是一个纯前端治愈系日记应用。你在写日记时，右下角的猫咪会认真倾听；保存成功时，它会开心地蹦跳；删除日记时，它会温柔地安慰你。所有数据保存在浏览器本地，无需注册、无需联网。

## 主要功能

- **日记编辑** — 标题（50字限制+实时计数）+ 正文，Ctrl/Cmd+S 快速保存
- **历史列表** — 按时间倒序展示，支持关键词搜索过滤
- **详情查看** — 模态框展示完整日记内容
- **二次确认删除** — 滑出动画后从存储删除，防止误删
- **猫咪动画系统** — 待机浮动、开心弹跳、难过倾斜、倾听点头四种状态
- **文字气泡** — 与猫咪动画同步，保存/删除/输入时随机显示治愈文案
- **时间问候** — 根据当前时段显示不同的欢迎气泡
- **周期闲聊** — 每30秒猫咪自动说一句话
- **响应式设计** — 适配桌面、平板、手机（320px+）
- **数据持久化** — LocalStorage 存储，刷新不丢失

## 技术栈

| 层级 | 技术 |
|------|------|
| 结构 | HTML5 语义化标签 |
| 样式 | CSS3（自定义属性、渐变、动画、媒体查询） |
| 逻辑 | 原生 JavaScript（ES Module） |
| 猫咪 | 内联 SVG |
| 存储 | localStorage |
| 构建 | Vite 5 |

### 基本操作

1. 打开页面后自动聚焦标题输入框
2. 输入标题和正文，点击「💾 保存日记」或按 Ctrl+S
3. 切换到「📖 历史记录」查看已保存日记
4. 点击「查看」打开详情模态框
5. 点击「删除」经二次确认后删除日记

## 设计特色

- **暖米色主色调** (#e8a87c) — 温柔治愈的视觉基调
- **渐变按钮** — 135deg 线性渐变 + hover 上浮动画
- **卡片阴影** — 带主色调的柔和投影
- **气泡过渡** — CSS opacity + transform 缓动淡入淡出
- **iOS Safari 兼容** — translateZ(0) GPU 合成、16px 输入框字号防放大
- **防重复提交** — 保存/删除按钮 disabled 冷却期
